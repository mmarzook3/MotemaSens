# Startup logo mask fix

Firmware: `dev-2026.06.13.22`

## Goal

Fix the missing section of the startup logo. The previous splash icon was drawn manually with line primitives and masking rectangles, which cut away part of the logo on the round display.

## Change

- Replaced the hand-drawn startup icon with an embedded 1-bit logo mask.
- Generated the mask from `docs/firmware/assets/motemasens_logo_source.svg`.
- Firmware draws the mask in the splash yellow foreground color.
- No runtime file loading is needed.

## Verification

- Built firmware with PlatformIO successfully.
- Flashed `dev-2026.06.13.22` to the device on `COM14`.
- Serial reset capture confirmed `device version: dev-2026.06.13.22`.
- Camera proof confirms the full icon is visible and the previously missing logo section is restored.
